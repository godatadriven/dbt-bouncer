# Tests

!!! note

    The below checks require `manifest.json` to be present.

These checks enforce test coverage on your dbt models. Use them to verify that every model has a unique test, that models meet a minimum number of unit tests, and that your overall project achieves a target test coverage percentage. Adequate test coverage catches data quality regressions early and gives your team confidence when refactoring or adding new models.

::: manifest.models.tests
