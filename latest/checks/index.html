    Checks

    dbt-bouncer runs checks against artifacts from dbt. These checks fall into three categories:

    • Catalog checks:
      • Catalog Sources
      • Columns
    • Manifest checks:
      • Exposures
      • Lineage
      • Macros
      • Metadata
      • Models
      • Semantic Models
      • Sources
      • Unit Tests
